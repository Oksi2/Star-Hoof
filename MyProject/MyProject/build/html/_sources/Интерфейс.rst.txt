Интерфейс
=========

#. Карта мира
#. Рюкзак
#. Список заданий
#. Кнопка для выхода из игры

Карта мира
----------
Мини-карта находится в правом нижнем углу игры.

Чтобы открыть полную карту мира, нужно нажать либо на мини-карту, либо на клавишу M.

Рюкзак
------
Иконка рюкзака находится рядом с мини-картой.

Чтобы открыть рюкзак, нужно нажать либо на иконку рюкзака, либо на клавишу I.

Список заданий
--------------
Иконка списка заданий находится рядом с мини-картой.
На самой иконке изображена книга.

Чтобы открыть список заданий, нужно нажать либо на иконку, либо на клавишу L.

В списке заданий показаны те задания, которые выполняет игрок в данный момент и те, которые доступны.

Кнопка для выхода из игры
-------------------------
Кнопка выхода из игры находится рядом с мини-картой.
На самой кнопке изображен крест.

Чтобы выйти из игры, нужно нажать либо на кнопку, либо на клавишу Esc.

Перед выходом из игры появится окно с подтверждением выхода.
